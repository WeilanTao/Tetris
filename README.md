# Tetris in C#
Classic Tetris game using WPF mvvm. Users can move and rotate tetraminos in a 10\*20 grid. Tetraminos falls automatically. There is a panel that displays the next tetramino.

## Tech Stack
WPF-MVVM, C#

## KeyBoard Setting
| Key |  |
|---|---|
| ↓ | Move Down |
| ← | Move left |
| → | Move right |
| A | Rotation left |
| D | Rotation right |
| W | HardDrop |
